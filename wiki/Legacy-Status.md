# Legacy Status

The existing wiki contains useful technical documentation, but much of it was
written for the old broad-feature strategy:

- many bots
- many loadouts
- wide feature coverage
- flashy tools
- wiki pages as broad user-facing strategy

Current strategy is narrower: one strong 1v1 PvP bot on `mc-26.1.2`.

## Rule

Source code defines runtime truth. Current strategy docs define product
direction. Wiki pages are reference until verified and rewritten.

## Legacy Reference Pages

Treat existing pages such as `Commands`, `Loadouts`, `Combat Behaviors`,
`Movement Network`, `Movement Brain Bank`, `AI Training`, `Brain Persistence`,
and `Configuration` as technical reference unless they explicitly say they have
been rewritten for the current strategy.

Do not delete legacy pages without preserving their content.
